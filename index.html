anyany.js

AI-powered CLI for QA: Bug Analysis, Scenario Prioritization, and Test Data Generation.
Supports OpenAI & Ollama (local LLMs).
  • Choose between OpenAI (cloud) or Ollama (local) models
  • Automatic structured prompts for bug, scenario, and test data tasks
  • Results saved in the output/ folder
  • Auto-detects and starts Ollama server if needed
  • Environment variables are safe – .env never committed
Quick Start
git clone https://github.com/modalqa/anyany.js.git
cd anyany.js
npm install
cp .env.example .env
# Edit .env and set your OPENAI_API_KEY if you want to use OpenAI
node agent.js

Example Prompts
# Bug Analyst
Analyze the following bug description or log and provide a structured output with:
- Bug Title
- Possible Cause
- Steps to Reproduce
- Expected Result
- Actual Result
- Impact Level (Low/Medium/High/Critical)
- Suggested Next Step

# Scenario Priority
Analyze the following test scenarios and assign a priority (High/Medium/Low) based on user impact, likelihood, and technical complexity. Provide logical justification for each.

# Test Data Generator
Generate test data for the following scenario. Include:
- Field Name
- Valid Data
- Invalid Data
- Edge Case Data
- Justification
Format the result as JSON.
